# Biblioteca API
Este projeto tem como objetivo principal a prática e aprimoramento dos estudos de APIs em .NET que venho realizado desde julho do ano passado. De lá pra cá, tudo que venho aprendendo, eu repliquei neste pequeno, mas muito importante para mim, projeto de um Sistema Bibliotecário.

Na V1 desse sistema, aquele que consumir a API será capaz de verificar Autores, Editoras, Categorias (ou Gêneros de livros), Livros, Usuários e Perfis de Usuário.
O projeto ainda está em desenvolvimento, assim como o README, e as próximas atualizações virão de forma periódica.
